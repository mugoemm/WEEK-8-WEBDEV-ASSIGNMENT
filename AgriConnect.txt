        <section>
            <div class="container">
                <div class="section-title">
                    <h2>Horticulture Tips</h2>
                    <p>Expert advice for growing healthy plants</p>
                </div>
                <div class="tips-container">
                    <div class="tip-card">
                        <h3>Growing Healthy Cabbages</h3>
                        <p>Cabbages thrive in well-drained soil rich in organic matter. Plant them in an area that receives at least 6 hours of sunlight daily. Keep the soil consistently moist and use natural pest control methods like neem oil to protect your plants.</p>
                    </div>
                    <div class="tip-card">
                        <h3>Capsicum Growing Guide</h3>
                        <p>Capsicums prefer warm temperatures and plenty of sunlight. Space plants about 18-24 inches apart to allow for proper air circulation. Harvest when fruits are firm and fully colored for the best flavor.</p>
                    </div>
                    <div class="tip-card">
                        <h3>Carrot Cultivation Tips</h3>
                        <p>For straight carrots, ensure loose, stone-free soil. Plant seeds directly in the garden as carrots don't transplant well. Thin seedlings to prevent overcrowding. Keep soil consistently moist for sweet, tender roots.</p>
                    </div>
                    <div class="tip-card">
                        <h3>Organic Pest Control</h3>
                        <p>Use companion planting to naturally repel pests. Marigolds deter nematodes, while basil helps protect tomatoes. Neem oil and insecticidal soap are effective organic treatments for common garden pests.</p>
                    </div>
                    <div class="tip-card">
                        <h3>Watering Best Practices</h3>
                        <p>Water deeply but infrequently to encourage deep root growth. The best time to water is early morning, which reduces evaporation and prevents fungal diseases. Use mulch to retain soil moisture.</p>
                    </div>
                    <div class="tip-card">
                        <h3>Soil Health Management</h3>
                        <p>Regularly add compost to improve soil structure and fertility. Practice crop rotation to prevent nutrient depletion and reduce disease buildup. Test your soil periodically to maintain optimal pH levels.</p>
                    </div>
                </div>
            </div>
        </section>

    <script>
        // Navigation functionality
        document.addEventListener('DOMContentLoaded', function() {
            const navLinks = document.querySelectorAll('.nav-link');
            const pageContents = document.querySelectorAll('.page-content');
            
            navLinks.forEach(link => {
                link.addEventListener('click', function(e) {
                    e.preventDefault();
                    const targetPage = this.getAttribute('data-page');
                    
                    // Hide all pages
                    pageContents.forEach(page => {
                        page.style.display = 'none';
                    });
                    
                    // Show target page
                    document.getElementById(targetPage).style.display = 'block';
                    
                    // Scroll to top
                    window.scrollTo(0, 0);
                });
            });
            
            // Form submission handling
            const bookingForm = document.getElementById('booking-form');
            if (bookingForm) {
                bookingForm.addEventListener('submit', function(e) {
                    e.preventDefault();
                    alert('Thank you for your booking! We will contact you shortly to confirm your order.');
                    bookingForm.reset();
                });
            }
            
            const contactForm = document.getElementById('contact-form');
            if (contactForm) {
                contactForm.addEventListener('submit', function(e) {
                    e.preventDefault();
                    alert('Thank you for your message! We will get back to you soon.');
                    contactForm.reset();
                });
            }
            
            // Show/hide product selection based on service type
            const serviceSelect = document.getElementById('service');
            const productSelection = document.getElementById('product-selection');
            const quantityGroup = document.getElementById('quantity-group');
            
            if (serviceSelect) {
                serviceSelect.addEventListener('change', function() {
                    if (this.value === 'product-order') {
                        productSelection.style.display = 'block';
                        quantityGroup.style.display = 'block';
                    } else {
                        productSelection.style.display = 'none';
                        quantityGroup.style.display = 'none';
                    }
                });
            }
            
            // Mobile menu toggle
            const mobileMenuBtn = document.querySelector('.mobile-menu-btn');
            const nav = document.querySelector('nav ul');
            
            if (mobileMenuBtn) {
                mobileMenuBtn.addEventListener('click', function() {
                    nav.style.display = nav.style.display === 'flex' ? 'none' : 'flex';
                });
            }
        });
    </script>
